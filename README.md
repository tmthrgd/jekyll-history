# jekyll-history

jekyll-history clones a git repository and runs `jekyll build` at any commit in its history.

## Download:

```
go get tmthrgd.dev/go/jekyll-history
```

## Run:

`jekyll-history [-safe=false] [-port=<port>] <repository url>`
